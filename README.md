# Job Assignment

### Requirments
1- PHP 8.1 <br>
2- Composer <br>
3- MySQL

### Execute Project 
1- Make clone on local computer <br>
2- Dump database.sql
3- Run composer install --ignore-platform-reqs in root directory <br>
4- Change env in Global.php
5- run php -S localhost:8080 -t ./public to run the project
